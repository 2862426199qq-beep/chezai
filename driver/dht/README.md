# DHT11 温湿度模块

## 当前状态

**软件模拟模式** — 由于 DHT11 硬件故障，当前采用软件模拟方式提供温湿度数据，未连接实际传感器。

## 版本迭代

| 版本 | 文件 | 说明 |
|------|------|------|
| v1 | test_dht11.c | 初版 GPIO 时序模拟 |
| v2 | test_dht11_v2.c | 时序优化 |
| v3 | test_dht11_v3.c | 容错改进 |
| v4 | test_dht11_v4.c | 稳定性增强 |
| v5 | test_dht11_v5.c | 最新版本 |

## 备注

- 协议：单线制 GPIO 软件模拟（非内核驱动）
- 待硬件修复后切换回真实传感器读取
